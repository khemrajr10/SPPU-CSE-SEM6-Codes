Practical11_word_count

Write a code in SCALA for a simple WordCount application that counts the number of 
occurrences of each word in a given input set using the Hadoop MapReduce framework on 
local-standalone set-up.

Code:

import scala.io.Source 
import scala.util.Using 

val counts = Using(Source.fromFile("/home/student/Downloads/data.txt")) { file => 

file.getLines() 
.toList 
.flatMap(_.split(" ")) 
.groupBy(identity) 
.view 
.mapValues(_.size) 
.toMap 

}.get 

counts.foreach(println)

Scala is a high-level programming language that combines:

Object-Oriented Programming (OOP)
Functional Programming (FP)

Scala runs on:

JVM (Java Virtual Machine)

just like Java.

Why Scala is Used?

Scala is used because:

code is shorter than Java
fast processing
supports big data frameworks
works well with Spark and Hadoop

Apache Hadoop is a framework used for:

storing large data
processing large data
distributed computing
Why Hadoop is Needed?

Normal systems cannot efficiently process:

Big Data

Example:

TBs of logs

Hadoop solves this using:

MapReduce is a programming model used to process large datasets in parallel.

It has 2 phases:

Phase	Work
Map	Processes input data
Reduce	Combines results
