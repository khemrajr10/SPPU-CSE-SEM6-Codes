Practical12_MapReduce

Design a distributed application using MapReduce which processes a log file of a system.

Code:

import scala.io.Source

val counts = Source.fromFile("log.txt")

.getLines()

.flatMap(_.split(" "))

.map(word => {

  if (word.startsWith("INFO"))
    "INFO"

  else if (word.startsWith("ERROR"))
    "ERROR"

  else if (word.startsWith("WARN"))
    "WARN"

  else
    "OTHER"
})

.filter(_ != "OTHER")

.toList

.groupBy(word => word)

.map {
  case (word, list) =>
    (word, list.size)
}

counts.foreach {
  case (k, v) =>

    print(s"($k,$v) ")
}

5. What is MapReduce?

MapReduce is a programming model used for processing large datasets in parallel.

It has two main phases:

Phase	Purpose
Map	Processes and transforms input
Reduce	Aggregates final result
6. Explain Map Phase.

Map phase:

reads input data
converts it into key-value pairs

Example:

Input:

INFO ERROR INFO

Map Output:

(INFO,1)
(ERROR,1)
(INFO,1)
7. Explain Shuffle and Sort Phase.

In this phase:

similar keys are grouped together

Example:

INFO -> [1,1]
ERROR -> [1]
8. Explain Reduce Phase.

Reduce phase:

combines grouped values
calculates final count

Output:

INFO -> 2
ERROR -> 1
