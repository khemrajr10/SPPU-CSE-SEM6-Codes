Practical13_WeatherData

Locate dataset (e.g., sample_weather.txt) for working on weather data which reads the text 
input files and finds average for temperature, dew point and wind speed. 

Code:

import scala.io.Source 
val data = Source.fromFile("weather.txt").getLines().toList 
var tempSum = 0.0 
var dewSum = 0.0 
var windSum = 0.0 
var count = 0 
for (line <- data) { 
val values = line.split(" ") 
tempSum += values(0).toDouble 
dewSum += values(1).toDouble 
windSum += values(2).toDouble 
count += 1 
} 
println("Average Temperature: " + tempSum / count) 
println("Average Dew Point: " + dewSum / count) 
println("Average Wind Speed: " + windSum / count)
